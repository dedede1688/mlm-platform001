```tsx
function AssetCard({
  label,
  value,
  color,
  desc,
}: {
  label: string
  value: number
  color: string
  desc: string
}) {
  const [textColor, bgColor] = color.split(' ')
  return (
    <div className="bg-white rounded-lg p-3 sm:p-4 border border-gray-100">
      <div className={`w-8 h-8 sm:w-9 sm:h-9 rounded-lg ${bgColor} ${textColor} flex items-center justify-center mb-1.5 sm:mb-2`}>
        <Wallet className="w-4 h-4 sm:w-5 sm:h-5" />
      </div>
      <p className="text-sm sm:text-lg font-bold text-gray-900">¥{formatMoney(value)}</p>
      <p className="text-[10px] sm:text-xs text-gray-500">{label}</p>
      <p className="text-[10px] text-gray-400 mt-0.5">{desc}</p>
    </div>
  )
}
```

**关键点**：
- 用 `Wallet` 图标（已经 import 在 line 7）
