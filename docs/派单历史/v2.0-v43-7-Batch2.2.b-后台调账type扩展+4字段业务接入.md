## 0. 任务背景

v43-7 Batch 2.2.a（commit 788de58）加了 4 个账户字段：
- `consume_balance` — 消费余额
- `earnings_pending` — 待结算收益（路径 1 下永远 0）
- `earnings_available` — 可提现收益
- `earnings_voided` — 累计作废

DB 已同步（已验证：4 列都在，默认 0）。但**业务代码 0 处引用**——4 字段是死账户。

**Batch 2.2.b 目标**：让管理员能通过后台调账 API 往 4 字段写值 + 后台/管理端 UI 能看到 4 字段。

**完整设计文档**：`docs/v43-7-design.md` 章节 8（胡子哥已拍板走路径 1：轻量叠加统计）

**改后代码**：
```ts
// 类型 → 字段映射表（v43-7 Batch 2.2.b 新增）
const TYPE_FIELD_MAP: Record<AdjustType, { main: 'balance' | 'frozenBalance'; extra?: 'consumeBalance' | 'earningsAvailable' | 'earningsVoided'; label: string }> = {
  balance:          { main: 'balance',          label: '余额' },
  frozenBalance:    { main: 'frozenBalance',    label: '冻结余额' },
  recharge:         { main: 'balance', extra: 'consumeBalance',     label: '余额/消费余额' },      // +amount
  consume_void:     { main: 'balance', extra: 'consumeBalance',     label: '余额/消费余额' },      // -amount
  earnings_add:     { main: 'balance', extra: 'earningsAvailable',  label: '余额/可提现收益' },    // +amount
  earnings_void:    { main: 'balance', extra: 'earningsVoided',     label: '余额/累计作废' },      // +amount（作废计正数）
}

// 注意：consume_void / earnings_add / earnings_void 的 main 字段都用 balance
//   - consume_void: balance -= amount, consumeBalance -= amount（消费扣减）
//   - earnings_add: balance += amount, earningsAvailable += amount（手动加可提现）
//   - earnings_void: balance -= amount, earningsVoided += amount（作废计 +|balance 扣 -|）
// extra 字段方向不一定是 +：consume_void 实际是 -，earnings_add 是 +，earnings_void 是 +
// 所以"加"和"扣"是分开的，由 type 决定，**不要**在 TYPE_FIELD_MAP 里用 sign
const TYPE_EXTRA_SIGN: Partial<Record<AdjustType, 1 | -1>> = {
  recharge: 1,
  consume_void: -1,
  earnings_add: 1,
  earnings_void: 1,  // 作废累加是正数（累加审计）
}

const result = await prisma.$transaction(async (tx) => {
  // 步骤 1：查旧值
  const before = await tx.user.findUnique({ where: { id } })
  if (!before || before.status === 'deleted') {
    throw new Error('用户不存在')
  }

  // 步骤 2：根据 type 决定改哪些字段
  const mapping = TYPE_FIELD_MAP[adjustType]
  const extraSign = TYPE_EXTRA_SIGN[adjustType] ?? 1  // balance / frozenBalance 默认 +1

  // 主字段 update
  const updateData: Record<string, { increment: number }> = {
    [mapping.main]: { increment: amount },
  }
  // 4 字段额外写入（如有）
  if (mapping.extra && extraSign !== undefined) {
    updateData[mapping.extra] = { increment: amount * extraSign }
  }

  // 防并发透支（仅主字段校验，4 字段不需要——它们是累加器）
  const mainWhereCond = amount < 0 ? { gte: Math.abs(amount) } : undefined
  const updateResult = await tx.user.updateMany({
    where: {
      id,
      ...(mainWhereCond ? { [mapping.main]: mainWhereCond } : {}),
    },
    data: updateData,
  })
  if (updateResult.count === 0) {
    throw new Error(`${mapping.label}不足`)
  }

  // 步骤 3：计算新值（用于 BalanceRecord 写入）
  const newBalance = mapping.main === 'balance' ? before.balance + amount : before.balance
  const newFrozenBalance = mapping.main === 'frozenBalance' ? before.frozenBalance + amount : before.frozenBalance
  const oldValue: Record<string, unknown> = {
    [mapping.main]: mapping.main === 'balance' ? before.balance : before.frozenBalance,
  }
  if (mapping.extra) {
    oldValue[mapping.extra] = (before as any)[mapping.extra] ?? 0
  }

  // 步骤 4：写 BalanceRecord
  //   4 字段调账 type 用 'admin_adjust'（与现有保持一致）
  //   description 标注 4 字段名（如"消费余额+50"）
  const extraDesc = mapping.extra
    ? `，${getFieldLabel(mapping.extra)}${amount > 0 ? '增加' : '扣减'} ¥${Math.abs(amount).toFixed(2)}`
    : ''
  await tx.balanceRecord.create({
    data: {
      userId: id,
      type: 'admin_adjust',
      amount,
      balance: newBalance,
      frozenBalance: newFrozenBalance,
      sourceType: 'admin',
      sourceId: admin.id,
      description: `管理员调账：${mapping.label}${amount > 0 ? '增加' : '扣减'} ¥${Math.abs(amount).toFixed(2)}${extraDesc}，原因：${reason}`,
    },
  })

  const updated = await tx.user.findUnique({ where: { id } })
  if (!updated) throw new Error('用户更新后查询失败')
  return { updated, oldValue, adjustType, mapping }
})
```

**新增 helper**（在文件底部、export 之前）：
```ts
function getFieldLabel(field: string): string {
  const labels: Record<string, string> = {
    consumeBalance: '消费余额',
    earningsPending: '待结算收益',
    earningsAvailable: '可提现收益',
    earningsVoided: '累计作废',
    balance: '余额',
    frozenBalance: '冻结余额',
  }
  return labels[field] ?? field
}
```

**关键设计**：
