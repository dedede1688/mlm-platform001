## 3. 改动 2：finance/page.tsx 操作列加"查看审核日志"链接

**位置**：操作列（最后 1 列）

**当前代码**：
```tsx
<td>... <button onClick={() => handleRefund(w)}>↩</button></td>
```

**改后代码**：
```tsx
<td>
  <div className="flex items-center gap-2">
    <button onClick={() => openAuditLogModal(w)} title="查看审核日志">
      <HistoryIcon className="w-4 h-4" />
    </button>
    {w.status === 'pending' && (
      <>
        <button onClick={() => handleApprove(w)}>✓</button>
        <button onClick={() => openRejectModal(w)}>✗</button>
      </>
    )}
  </div>
</td>
```

**Audit log modal**：
```tsx
<Modal open={auditLogOpen} onClose={closeAuditLogModal} title="审核日志">
  <div className="space-y-3">
    {auditLogs.map(log => (
      <div key={log.id} className="border-b pb-2">
        <div className="flex justify-between text-sm">
          <span className="font-medium">{log.admin.nickname} ({log.admin.phone})</span>
          <span className="text-gray-500">{formatDate(log.createdAt)}</span>
        </div>
        <div className="text-sm mt-1">
          动作: <span className="font-medium">{log.action}</span>
          {' | '}
          {log.oldStatus} → {log.newStatus}
        </div>
        {log.reason && <div className="text-sm">理由: {log.reason}</div>}
        {log.remark && <div className="text-sm text-gray-600">备注: {log.remark}</div>}
      </div>
    ))}
  </div>
</Modal>
```

**API 调用**：
```typescript
const openAuditLogModal = async (withdrawal) => {
  setAuditLogLoading(true)
  const res = await fetch(`/api/admin/withdrawals/${withdrawal.id}/audit-logs`)
  const data = await res.json()
  setAuditLogs(data.logs || [])
  setAuditLogOpen(true)
  setAuditLogLoading(false)
}
```
