## 规则优先级

1. 系统和用户的直接指令。
2. 项目根 `AGENTS.md`。
3. 本目录的岗位文件。
4. 当前任务提示词。
5. 历史归档只用于审计，不是现行规则。

## 派单方式（2026-07-22 胡子老师拍板）

- **提示词不落文件**：小酷在对话中直接输出"可一键复制"的完整任务提示词，胡子老师复制后发给小猫/小M 的独立会话执行。
- 提示词必须自包含：小猫/小M 无法反问、无法查小酷的对话历史。
- 模板参考：`docs/roles/templates/`（task / result / review 三种格式）。
