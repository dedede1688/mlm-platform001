# 猫爪_039号任务：清理垃圾文件 + 任务文档纳入 git

> 执行角色：猫爪辅助执行位
> 任务性质：文件清理 + 文档管理
> 工作方式：分两个阶段，每阶段精确暂存
> 结果文件：`docs/agent-tasks/catpaw/done/猫爪_039号结果.md`

## 1. 任务目标

1. 删除项目根目录的垃圾文件。
2. 把历史临时脚本移到归档目录。
3. 把所有猫爪和小 M 的任务文件、结果文件、岗位说明纳入 git。

## 2. 允许修改的文件

### 阶段一：清理

- 删除：`vercel_ls.txt`（根目录调试导出文件）
- 移动：`scripts/seed-test-amount-confirm.ts` → `docs/归档/scripts/`
- 移动：`scripts/cleanup-test-amount-confirm.ts` → `docs/归档/scripts/`

### 阶段二：文档入库

- `git add docs/agent-tasks/`（所有 catpaw 和 xiaom 的任务文件、结果文件）
- `git add docs/superpowers/`（设计文档和实施计划）

## 3. 禁止操作

- 不修改任何 `src/` 下的文件。
- 不修改任何 `__tests__/` 下的文件。
- 不修改 `AGENTS.md`、`prisma/schema.prisma`、`package.json`。
- 不碰已提交的业务代码。
- 阶段一和阶段二分别 commit，不混在一起。

## 4. 执行步骤

### 阶段一：清理

```powershell
# 1. 删除垃圾文件
Remove-Item vercel_ls.txt

# 2. 创建归档目录（如不存在）
New-Item -ItemType Directory -Path docs/归档/scripts -Force

# 3. 移动临时脚本
Move-Item scripts/seed-test-amount-confirm.ts docs/归档/scripts/
Move-Item scripts/cleanup-test-amount-confirm.ts docs/归档/scripts/

# 4. 精确暂存
git add vercel_ls.txt docs/归档/scripts/
git rm --cached scripts/seed-test-amount-confirm.ts scripts/cleanup-test-amount-confirm.ts 2>$null

# 5. 确认暂存区
git diff --cached --name-only
```

### 阶段二：文档入库

```powershell
# 1. 暂存所有任务文档
git add docs/agent-tasks/
git add docs/superpowers/

# 2. 确认暂存区只有 docs 文件
git diff --cached --name-only
```

## 5. 验证命令

```bash
# 确认工作区干净（除了已提交的文件）
git status --short

# 确认没有 src/ 或 __tests__/ 文件被暂存
git diff --cached --name-only | findstr "src/"
git diff --cached --name-only | findstr "__tests__/"
```

## 6. 完成标准

1. `vercel_ls.txt` 已删除。
2. 临时脚本已移到 `docs/归档/scripts/`。
3. 所有 `docs/agent-tasks/` 和 `docs/superpowers/` 下的文件已纳入 git。
4. 暂存区没有任何 `src/` 或 `__tests__/` 文件。
5. 两个阶段分别 commit，commit message 清楚说明做了什么。
6. `git status --short` 中不再有 `docs/agent-tasks/` 的 untracked 文件。
