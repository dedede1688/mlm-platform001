## 四、最小实现

只修改 `handleEarningsTransferSuccess` 内的 `setPendingPayments`（更新待支付状态）部分。

推荐写法：

```typescript
setPendingPayments(prev => {
  const current = prev[checkoutItem.id]
  if (!current) return prev

  return {
    ...prev,
    [checkoutItem.id]: {
      ...current,
      shortage: remainingShortage,
    },
  }
})
```

必须满足：

- 记录不存在时不新增任何键。
- 不生成空 `orderId`。
- 保留当前记录真实 `orderId`。
- 只更新当前商品 `shortage`。
- 保留其他商品记录。
- 不自动调用支付。
- 不改变用户资金刷新和提示文案。
