## 9. 状态重置函数和统一关闭函数如何实现

### resetPaymentPasswordResetState

```typescript
const resetPaymentPasswordResetState = () => {
  setPayPwdResetReason('')
  setPayPwdResetSuffix('')
  setShowPayPwdConfirm(false)
}
```

- 清空支付密码重置原因、手机号后 4 位、二次确认弹窗状态
- 不触碰其他无关状态（balanceReason、pointsReason 等）

### closeDetailModal

```typescript
const closeDetailModal = () => {
  resetPaymentPasswordResetState()
  setDetailUser(null)
}
```

- 先清空支付密码重置状态，再关闭详情弹窗
- 被三处调用：遮罩 onClick、右上角 X 按钮、底部"关闭"按钮
