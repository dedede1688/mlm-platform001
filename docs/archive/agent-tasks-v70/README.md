# v70智能体任务历史归档

本目录保存角色制度硬重置前的任务、结果和审核记录，仅用于审计与问题追溯。

## 使用规则

- 本目录不是当前岗位制度。
- 不从本目录派发新任务。
- 不把本目录内的岗位描述当作现行规则。
- 历史正文、编号和结论保持原样。
- 当前制度统一以项目根 `AGENTS.md` 和 `docs/roles/` 为准。
