## 六、`earnings_add` 是否仍然正确提交

✅ 是。

资金调整选择「可用收益」时，`balanceType` 状态值为 `'earnings_add'`，提交给后端的 `type` 字段为 `earnings_add`。

代码路径：
```typescript
const [balanceType, setBalanceType] = useState<'balance' | 'frozenBalance' | 'earnings_add'>('balance')
// ...
body: JSON.stringify({ type: balanceType, amount, reason: balanceReason.trim() }),
```
