File: D:/mlm-platform-source/mlm-platform/__tests__/__tmp_diag_jsx.test.ts

### 5.3 5 个层级的逐一排除

| 层级 | 是否根因 | 证据 |

| 外部模块副作用 | ❌ | 错误在 import 阶段，未到模块执行 |
| DOM 环境（`environment: 'node'`） | ❌ | `renderToStaticMarkup` 是 React 内部纯函数，不依赖 DOM；`environment: 'node'` 不会让 React 报错 |

**因此 jsdom 完全是错误方向**。

**Vite 8 强制使用 oxc（Rust 实现）**：
- 更快的转译速度
- 但当前**不支持 JSX**
- 配置 esbuild 的 `jsx: 'automatic'` **被 oxc 显式忽略**

**Vite 7 及之前**默认使用 esbuild：
- 支持 JSX（jsx: 'automatic'）
- 但 Vite 8 已强制迁移到 oxc
