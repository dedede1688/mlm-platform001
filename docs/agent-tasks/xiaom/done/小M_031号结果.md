**实际代码**（L358-365）：

```typescript
function getButtonByText(html: string, text: string): string {
  const buttons = [...html.matchAll(/<button\b[^>]*>[\s\S]*?<\/button>/g)]
    .map((match) => match[0])
  const matchingButtons = buttons.filter((button) => button.includes(text))

  expect(matchingButtons, `应唯一找到按钮：${text}`).toHaveLength(1)
  return matchingButtons[0]
}
```

**结构分析**：
1. `matchAll(/<button\b[^>]*>[\s\S]*?<\/button>/g)` 枚举所有 `<button>` 元素
2. `.map((match) => match[0])` 取每个完整 button 字符串
3. `.filter((button) => button.includes(text))` 按文字筛选
4. `expect(matchingButtons, ...).toHaveLength(1)` 强制恰好 1 个
5. 返回唯一的 button 字符串

**正则 `[\s\S]*?` 仅在单 button 内部**——`<button ...>...</button>` 配对由 `matchAll` 的非贪婪保证，**不会跨标签**。

**禁止 `if (match)` 静默跳过**：`toHaveLength(1)` 是严格断言，0 个或 >1 个都失败。

**禁止 `A || B` 任一路径通过**：无 OR 断言。

**已删除旧跨标签正则**（grep 验证 0 匹配）。

**P0-C 已闭合**。
